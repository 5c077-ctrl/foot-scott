# foot-scott

A polished football match prototype built in Python with Pygame as a native desktop application.

It includes:
- immersive stadium-style 2D presentation
- animated player movement and ball physics
- scoring, stamina, possession, and match clock
- menu flow and polished HUD
- packaged desktop executables in the dist folder: app.exe and setup.exe

This project is intended as a native desktop football experience rather than a web app.
